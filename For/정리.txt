For문
사용법-> for(초기식; 조건식; 변화식)
         {
            실행문1
            실행문2
            실행문3
         }
예~>     for(i = 10; i >= 1; i--)
         {
            printf("%d ", i);
            printf("초");
            printf("\n");
         }
쓰레기(garbage) 값 -> 변수 생성한 후, 그 변수에 들어있는 값
변수에 값을 넣는 방법-> init, assign
init = 초기화, 선언시 값을 넣어주면
assign = 대입, 선언시 이 외의 경우
두 경우 모두 대입연산자를 사용한다.
지역변수, 전역변수
지역변수: 자신만의 공간을 갔는 것
*어디에 있는지에 따라가 중요함.(예) for문 안에서 생성된 변수는 for문이 끝날 때 사라진다.
전역변수: 특정한 공간에 있어 어디에서나 사용할 수 있는 변수
